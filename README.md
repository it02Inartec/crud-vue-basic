# crud-vue-basic

This app is for create, read, uptate and delete clients on table with the data "name", "email" y "phone". 
Code from API https://github.com/it02Inartec/api-crud-laravel7


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```
